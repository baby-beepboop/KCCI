# ⚙️ 버튼 디바운스
### 개요
- 버튼을 누르고 뗄 때 발생하는 약 10ms의 노이즈를 **Timer counter를 활용하여** 디바운싱
- 버튼을 한 번 눌렀을 때 LED on, 다시 한 번 눌렀을 때 LED off
